# เริ่มต้นใช้งาน

หน้านี้สำหรับพนักงานที่ใช้งานโปรแกรม FindEndPromise

## เปิดโปรแกรม

1. เปิด `FindEndPromise.exe`
2. เข้าสู่ระบบด้วยชื่อผู้ใช้และรหัสผ่านตามปกติ
3. โปรแกรมจะแสดงข้อมูล Version ที่หน้า Login

## ถ้าไม่มีเวอร์ชันใหม่

ระบบจะแสดงข้อความประมาณนี้

```text
โปรแกรมเป็นเวอร์ชันล่าสุดแล้ว rev.144
```

สามารถกดปุ่ม **What's New?** เพื่อดูรายละเอียดการเปลี่ยนแปลงของเวอร์ชันปัจจุบันได้

## ถ้ามีเวอร์ชันใหม่

ระบบจะแสดงข้อความ เช่น

```text
พบโปรแกรมเวอร์ชันใหม่
Current : rev.141
Latest  : rev.144
```

พร้อมปุ่ม **Update Now**

## ขั้นตอน Update

1. กด **Update Now**
2. โปรแกรมหลักจะปิดตัวเอง
3. โปรแกรม `FindEndPromiseUpdater.exe` จะเปิดขึ้นมา
4. รอให้ Progress ถึง 100%
5. โปรแกรม FindEndPromise จะเปิดขึ้นใหม่อัตโนมัติ

## ข้อห้ามระหว่าง Update

- ห้ามปิดเครื่อง
- ห้าม Restart Windows
- ห้ามดึงสาย LAN หรือปิด Network
- ห้ามเปิด FindEndPromise ซ้ำ
- ห้ามลบไฟล์ในโฟลเดอร์โปรแกรม

## ถ้า Update ไม่สำเร็จ

ให้เปิดไฟล์สำรองในโฟลเดอร์เดียวกับโปรแกรม เช่น

```text
FindEndPromise-v142.exe
```

แล้วแจ้งฝ่าย IT พร้อมข้อความ Error ที่แสดงบนหน้าจอ
