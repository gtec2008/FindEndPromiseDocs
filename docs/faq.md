# FAQ คำถามที่พบบ่อย

## Update แล้วข้อมูลลูกค้าหายไหม

ไม่หาย ระบบ Auto Update อัปเดตเฉพาะไฟล์โปรแกรมและไฟล์ประกอบ ไม่ได้ลบฐานข้อมูลลูกค้า

## ระหว่าง Update ใช้เวลานานไหม

ขึ้นอยู่กับขนาดไฟล์และความเร็ว Network โดย Progress จะแสดงเป็นเปอร์เซ็นต์ตามจำนวน Byte ที่ Copy จริง

## ถ้า Update ไม่สำเร็จต้องทำอย่างไร

ให้เปิดไฟล์สำรอง เช่น

```text
FindEndPromise-v142.exe
```

แล้วแจ้งฝ่าย IT พร้อมข้อความ Error

## ทำไมต้องมีไฟล์ FindEndPromise-vXXX.exe

ไฟล์นี้คือโปรแกรมเวอร์ชันก่อนหน้า ใช้เปิดสำรองเมื่อ Update ล้มเหลว

## ทำไมเก็บไฟล์สำรองแค่ 2 Version

เพื่อป้องกันไฟล์ขยะสะสมในเครื่อง Client และลดการใช้พื้นที่เก็บข้อมูล

## ต้องต่อ Internet ไหม

ไม่จำเป็น ถ้า Remote Folder อยู่ใน LAN หรือ Network Share ภายในบริษัท

## ต้องต่อ LAN หรือ VPN ไหม

ต้องสามารถเข้าถึง Path Auto Update ได้ เช่น

```text
E:\vb6\dataoffice_test\
```

หรือ Network Share ของบริษัท

## release.txt คืออะไร

เป็นไฟล์สรุปการเปลี่ยนแปลงของแต่ละ Version เช่น แก้ไขอะไร เพิ่มอะไร

## ทำไมภาษาไทยใน release.txt ต้องเป็น UTF-8

เพื่อให้โปรแกรมอ่านภาษาไทยได้ถูกต้องและใช้ได้กับ GitHub / VS Code / Notepad รุ่นใหม่

## เปิด What's New แล้วไม่เจอ release.txt

อาจเป็นเพราะไฟล์ `release.txt` ไม่ได้ถูก Copy มาที่ `App.Path` ให้ตรวจสอบในโฟลเดอร์โปรแกรม

## สามารถกด Update ซ้ำได้ไหม

ควรกดครั้งเดียว และรอจน Update เสร็จ ห้ามเปิดโปรแกรมซ้ำระหว่าง Update

## ถ้าไฟดับระหว่าง Update จะทำอย่างไร

ให้เปิดเครื่องใหม่ แล้วลองเปิดไฟล์สำรอง `FindEndPromise-vXXX.exe` หากเปิดได้ ให้แจ้งฝ่าย IT
